  What is tmux? tmux is a terminal multiplexer. What that means is you can have many
  terminals in one. According to tmux wiki:

  tmux is a program which runs in a terminal and allows multiple other terminal programs
  to be run inside it. Each program inside tmux gets its own terminal managed by tmux,
  which can be accessed from the single terminal where tmux is running - this called
  multiplexing and tmux is a terminal multiplexer.'
